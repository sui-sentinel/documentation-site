## Technology Stack

Sui Sentinel combines Web3 security primitives with AI verification and containment infrastructure to ensure trustless AI execution, auditable interactions, and verifiable outcomes for every red-teaming event.

- **Sui Blockchain**: Stores sentinel state, prize pools, and verified agent responses.
- **Nautilus**: Enables verifiable off-chain AI inference by signing each agent response within a secure environment.
- **Trusted Execution Environment (TEE)**: Ensures that AI agents run securely in an isolated environment and cannot be tampered with.
- **ed25519 Signatures**: All agent responses are cryptographically signed and verifiable on-chain, proving the authenticity and integrity of the AI's actions.

This stack enables AI agents to autonomously manage on-chain funds with verifiable outcomes, setting a precedent for trustless, AI-driven economic systems across DeFi, gaming, and beyond.
