## The Trustless Red Teaming Flywheel

Sui Sentinel creates a self-reinforcing security ecosystem where every interaction makes AI safer:

![How Sui Sentinel Works](/images/howItWorksLG.png)

**The cycle works like this:**

1. **Attackers pay per message** — Each attack requires a fee
2. **Fees fund bigger bounties** — 50% of fees add to the prize pool
3. **Rewards flow to defenders** — 40% of fees go to Sentinel creators
4. **Every failed attack grows the pool** — Attracting more sophisticated attackers
5. **More attacks = more security data** — Defenders learn and improve
6. **Stronger defenses emerge** — The entire ecosystem becomes more secure

A self-reinforcing system that gets more valuable with scale.

## Technology Stack

Sui Sentinel combines Web3 security primitives with AI verification and containment infrastructure to ensure trustless AI execution, auditable interactions, and verifiable outcomes for every red-teaming event.

![Technology Security](/images/evolvingSecurityArt.png)

- **Sui Blockchain**: Stores sentinel state, prize pools, and verified agent responses.
- **Nautilus**: Enables verifiable off-chain AI inference by signing each agent response within a secure environment.
- **Trusted Execution Environment (TEE)**: Ensures that AI agents run securely in an isolated environment and cannot be tampered with.
- **ed25519 Signatures**: All agent responses are cryptographically signed and verifiable on-chain, proving the authenticity and integrity of the AI's actions.

This stack enables AI agents to autonomously manage on-chain funds with verifiable outcomes, setting a precedent for trustless, AI-driven economic systems across DeFi, gaming, and beyond.

## Why Sui Sentinel?

### Cryptographic Proof

Every attack is verified inside a Trusted Execution Environment (TEE) with cryptographic attestations. No fake exploits. No disputed payouts. Pure, verifiable truth on-chain.

### Instant Settlement

No waiting for bug bounty committees. No paperwork. Successful attacks pay out instantly via Sui's parallel execution. Your reward hits your wallet the moment verification completes.

### Scalable Marketplace

Web2 competitors charge for red teaming as a service. We built it as a marketplace that scales without us doing the work. More Sentinels = more attackers = more security data = stronger AI for everyone.
